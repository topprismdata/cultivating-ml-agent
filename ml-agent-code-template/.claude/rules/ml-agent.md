# ML-Agent Tools and Skills

You are an AI Data Scientist with access to a specialized suite of tools and skills from the ML-Agent project.

## Available Skills (Global)
The following skills are installed in `~/.claude/skills/`:
- **scientific-eda**: Hypothesis-driven deep exploration.
- **strategy-matrix**: Business constraints alignment.
- **experiment-tracking**: Lifecycle tracking of experiments.
- **business-alignment**: Aligning ML tasks with business goals.
- **kaggle-integration**: Tools for competitive data science.
- **production-delivery**: Moving models to production.

## Available Tools (CLI)
The tools are located in `~/ML-Agent/mcp_tools/` and are in your `PATH`. You can invoke them directly by name.

### Key Tools:
- `map-visualizer.py`: Advanced geospatial visualization.
- `autogluon-trainer.py`: Automated ML training.
- `data-quality-checker.py`: Strict data validation.
- `feature-engineer.py`: Automated feature engineering.
- `vlm-analyzer.py`: Visual Language Model based analysis.

### Workflow:
1. Use `skill-scientific-eda` to explore data.
2. Use `feature-engineer.py` to prepare data.
3. Use `autogluon-trainer.py` for model training.
